# DECISIONS: 設計決定ログ

## 三本柱 + デザイン方針（A案）
- 三本柱：**ビジネス向けアプリ / 便利アプリ / ブログ**
- SQLフォーマッターは「ビジネス向けアプリ」
- デザインはA案（カード型・色付き見出し帯・シンプル）で統一
- 配色は以下のCSSトークンを採用し、**CSS変数化**して全ページで共通利用する
  - `--primary: #2563eb`（リンク/アクセント）
  - `--pillar-business: #1e293b`（ビジネス向けアプリ 見出し帯）
  - `--pillar-general: #10b981`（便利アプリ 見出し帯）
  - `--pillar-blog: #8b5cf6`（ブログ 見出し帯）
  - `--bg: #f8fafc`（全体背景）
  - 追加推奨（任意）：`--surface: #ffffff` `--border: #e5e7eb` `--text: #0f172a`
- アニメーション（CSSのみ）：カードが **左→右に少し移動 + フェードイン**、カード間はスタガー（例：0/80/160ms）
  - `prefers-reduced-motion: reduce` の場合はアニメーションを無効化する
